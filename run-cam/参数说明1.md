图像基础调整

亮度 (Brightness) — 当前值=0，调节画面明暗，范围 -64（最暗）到 64（最亮）ok
对比度 (Contrast) — 当前值=0，调节明暗差异，范围 0～95，值越大反差越强 
饱和度 (Saturation) — 当前值=64，调节色彩鲜艳程度，值越大颜色越浓郁 
色调 (Hue) — 当前值=0，整体偏移颜色倾向（如偏红/偏绿），范围 -2000～2000 ok

白平衡

自动白平衡 (White Balance, Automatic) — 当前=开启(1)，自动校正色温偏差 ok
这个是 8 白平衡色温 (White Balance Temperature) — 仅在自动白平衡关闭时生效 ok

画质细节

伽马 (Gamma) — 当前值=100，控制亮度的非线性响应曲线，影响中间调的明暗表现 ok
锐度 (Sharpness) — 当前值=2，调节边缘锐利程度，范围 1～7，值越大轮廓越清晰 
背光补偿 (Backlight Compensation) — 当前=开启(1)，逆光场景下防止主体过暗 

曝光

电源频率 (Power Line Frequency) — 当前值=2（即 60Hz），0=禁用，1=50Hz，2=60Hz
自动曝光 (Auto Exposure) — 当前值=3（全自动），控制曝光模式（不同数值对应手动/光圈优先/快门优先/全自动）
曝光时间 (Exposure Time, Absolute) — 当前值=166 手动模式下控制快门速度，仅在关闭自动曝光时生效
